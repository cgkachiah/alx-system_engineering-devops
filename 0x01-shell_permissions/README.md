# Permissions Shell Project
###### 0. My name is Betty  ######
```
su betty
```
###### 1. Who am I ######
```
whoami
```
###### 2. Groups ######
```
groups
```
###### 3. New Owner ######
```
chwon
```
###### 4. Empty ######
```
touch
```
###### 5. Execute ######
```
chmod u+x hello
```
###### 6. Multiple Permissions ######
```
chmod ug+x,o+r hello
```
###### 7. Everybody ######
```
chmod ugo+x hello
```
